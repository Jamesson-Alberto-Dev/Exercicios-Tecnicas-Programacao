# 💻 Exercícios - Técnicas de Programação

Este repositório contém os códigos, listas de exercícios e projetos práticos desenvolvidos durante a disciplina de **Técnicas de Programação**. 

---

## 🎯 Objetivo

O foco principal destas atividades é aprofundar o conhecimento em lógica de programação, estruturação de algoritmos e aplicação de boas práticas no desenvolvimento de software utilizando a linguagem Java.

---

## 📚 Tópicos Abordados

Ao longo dos exercícios e listas (como a `lista1JAMESSON`), foram trabalhados conceitos fundamentais como:
* **Estruturas de Controle**: Condicionais (`if/else`, `switch`) e laços de repetição (`for`, `while`).
* **Estruturas de Dados Básicas**: Manipulação de vetores e matrizes (Arrays).
* **Lógica Aplicada**: Resolução de problemas matemáticos e cotidianos através de algoritmos.
* **Organização**: Modularização de código e uso da classe `Scanner` para interação via terminal.

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem**: Java
- **IDE**: Eclipse
- **Versionamento**: Git e GitHub
